# Algoritmos y Programación 
Este es el ABC de la programación. Empiezas sin saben absolutamente nada para adentrarte a profundidad en el mundo de la programación, algoritmos, resolución de problemas y razonamiento lógico del mismo.
## Se tratan temas como:
- Acciones elementales
- Estructuras de control condicional
- Estructuras de control iterativo
- Arreglos y matrices
- Programación orientada a objetos (POO)

# Algoritmos y Estructuras de datos
Habiendo dominado todo lo de **AyP** se encuentran con una nueva materia donde entenderan como se crean, manejan y estructuran los datos a tu conveniencia y otras técnicas de programacion.
## Se tratan temas como:
- Recursividad
- Backtracking
- Listas, pilas y colas
- Arboles
